## 1. 产品概述
A股量化交易预测工具——一款面向个人投资者的Web应用，通过输入股票代码，基于技术指标规律进行量化模拟交易，并预测股票下一阶段走势。
- 解决问题：帮助散户投资者快速获取技术面分析，辅助决策
- 目标用户：A股市场个人投资者、量化交易学习者

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 浏览和使用全部功能 |

### 2.2 功能模块
1. **行情分析页**：股票搜索、K线图表、技术指标叠加、预测信号展示
2. **量化模拟页**：策略选择、模拟交易回测、收益曲线、交易记录

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 行情分析页 | 股票搜索栏 | 输入A股代码/名称，自动补全匹配，加载股票数据 |
| 行情分析页 | K线图表区 | 展示日K/周K/月K，支持缩放拖拽，叠加MA/布林带/MACD/KDJ指标 |
| 行情分析页 | 趋势预测面板 | 基于多指标综合评分，显示看涨/看跌/震荡信号及置信度 |
| 行情分析页 | 关键价位提示 | 显示支撑位、压力位、止损位、止盈位 |
| 量化模拟页 | 策略选择器 | 预设多种量化策略（均线交叉、MACD金叉、布林带突破、KDJ超买超卖、多因子组合） |
| 量化模拟页 | 模拟回测面板 | 展示策略回测收益曲线、年化收益、最大回撤、夏普比率 |
| 量化模拟页 | 交易记录表 | 列出模拟买卖点、盈亏情况、持仓状态 |
| 量化模拟页 | 下一步操作建议 | 根据当前行情和选定策略，给出买入/卖出/持有建议 |

## 3. 核心流程

用户输入股票代码 → 系统获取历史行情数据 → 计算技术指标 → 展示K线与指标叠加图 → 运行量化策略模拟 → 输出预测信号与操作建议

```mermaid
    "展示K线图+指标叠加" --> "运行量化策略模拟"
    "运行量化策略模拟" --> "输出预测信号"
    "输出预测信号" --> "给出操作建议"
    "运行量化策略模拟" --> "展示回测结果"
    "展示回测结果" --> "交易记录与收益曲线"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深色金融风格——深蓝黑(#0a0e27)为底，荧光绿(#00ff88)代表涨，红色(#ff4757)代表跌
- 辅助色：金色(#ffd700)突出关键数据，冷蓝(#1e90ff)用于信息性元素
- 按钮风格：圆角胶囊型，渐变发光效果
- 字体：数字用等宽字体 JetBrains Mono，标题用思源黑体/Noto Sans SC
- 布局风格：深色仪表盘式，信息密集但层次分明，左侧导航+右侧主内容区
- 图标风格：线性图标，荧光描边效果

### 4.2 页面设计概述
| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 行情分析页 | 股票搜索栏 | 顶部固定搜索框，玻璃拟态背景，输入时下拉补全列表，荧光边框聚焦效果 |
| 行情分析页 | K线图表区 | 占主区域70%，深色画布，绿色阳线红色阴线，半透明指标线，网格线极淡 |
| 行情分析页 | 趋势预测面板 | 右侧卡片，大号信号文字（看涨/看跌/震荡），环形置信度指示器，渐变背景 |
| 行情分析页 | 关键价位提示 | 横向标签组，金色高亮价位数字，虚线标注在K线图上 |
| 量化模拟页 | 策略选择器 | 卡片式选择，选中态发光边框，策略参数可折叠调整 |
| 量化模拟页 | 模拟回测面板 | 双轴图表（收益曲线+基准线），下方统计指标网格 |
| 量化模拟页 | 交易记录表 | 深色表格，买入行绿色高亮，卖出行红色高亮，悬浮详情 |
| 量化模拟页 | 下一步操作建议 | 底部固定操作栏，大号操作按钮（买入/卖出/持有），脉冲动画吸引注意 |

### 4.3 响应式设计
- 桌面优先（1920x1080最佳），平板自适应（768px+），手机端简化布局（图表缩略+关键信息卡片）

### 4.4 数据来源说明
- 使用模拟/随机生成的A股历史数据（基于真实A股波动特征），不依赖外部API
- 股票列表预设50只热门A股代码和名称，用于搜索补全
